Introduction to NodeJS

Node.js is a popular and powerful open-source, cross-platform, back-end JavaScript runtime environment.

Created in 2009 by Ryan Dahl, it has quickly gained popularity among developers for its ability to run JavaScript on the server-side.

Node.js is built on Chrome's V8 JavaScript engine, which provides a fast, efficient, and scalable platform for building web applications.

Installing NodeJS

To install Node.js, go to the official website (https://nodejs.org/) and download the installer for your operating system.

Follow the installation instructions to complete the installation process.

Once installed, you can check the version of Node.js by running the following command in your terminal:

node -v
Creating a NodeJS Project

To create a new Node.js project, create a new directory and navigate into it using the terminal.

Once inside the directory, run the following command to initialize a new Node.js project:

npm init

Follow the prompts to set up the project, such as setting the project name, version, and description.

Using Built-in Modules in NodeJS

Node.js provides a set of built-in modules that can be used to perform various tasks.

These modules can be accessed using the `require` function and can beused in your code to simplify development.

Here are some examples of built-in modules in Node.js:

  • fs module: used to perform file system operations such as reading and writing files. Example code:
    • const fs = require('fs'); const data = fs.readFileSync('file.txt', 'utf-8'); console.log(data);
  • http module: used to create an HTTP server. Example code:
    • const http = require('http'); const server = http.createServer((req, res) => { res.end('Hello, World!'); }); server.listen(3000, () => { console.log('Server listening on port 3000'); });
  • path module: used to manipulate file paths. Example code:
    • const path = require('path'); const filePath = '/path/to/file.txt'; const fileName = path.basename(filePath); console.log(fileName);
    Using NPM Packages in NodeJS

    NPM (Node Package Manager) is the default package manager for Node.js.

    NPM allows developers to easily install and manage third-party packages.

    Here are some examples of popular NPM packages:

  • express: a fast, minimalist web framework for Node.js. Example code:
    • const express = require('express'); const app = express(); app.get('/', (req, res) => { res.send('Hello, World!'); }); app.listen(3000, () => { console.log('Server listening on port 3000'); });
  • moment: a library for parsing, validating, manipulating, and formatting dates. Example code:
    • const moment = require('moment'); const now = moment(); console.log(now.format('MMMM Do YYYY, h:mm:ss a'));
    Asynchronous Programming in NodeJS

    Node.js is designed to be asynchronous, meaning that it can handle multiple requests at the same time without blocking the main thread.

    Asynchronous programming in Node.js is typically done using callbacks, promises, or async/await.

    Here are some examples of asynchronous programming in Node.js:

  • Callbacks:
    • const fs = require('fs'); fs.readFile('file.txt', 'utf-8', (err, data) => { if (err) { console.error(err); } else { console.log(data); } });
  • Promises:
    • const fs = require('fs').promises; fs.readFile('file.txt', 'utf-8') .then(data => { console.log(data); }) .catch(err => { console.error(err); });
  • Async/await:
    • const fs = require('fs').promises; async function readFile() { try { const data = await fs.readFile('file.txt', 'utf-8'); console.log(data); } catch (err) { console.error(err); } } readFile();

    In summary, Node.js is a powerful and popular server-side platform for building web applications. It provides a set of built-in modules and a vast ecosystem of third-party packages through NPM. Asynchronous programming in Node.js is essential for handling multiple requests and can be done using callbacks, promises, or async/await. With its speed, scalability, and flexibility, Node.js is a valuable tool for developers of all levels.
